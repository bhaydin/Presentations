# Presentations

A place for me to share presentations that I have done, along with code snippets and samples that support them.

## Organization

This repository is organized by **year** and then by **event**:

```
<year>/
  <EventName>/
    README.md       ← event details and session descriptions
    *.pdf           ← presentation slide decks
    <topic>/        ← optional sub-folders for code samples
```
